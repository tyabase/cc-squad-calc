# C.C. Squad Calculator

A Complete Mortar Calculator for Squad - 战术小队火炮计算器

## 🌐 在线访问

**https://your-vercel-app.vercel.app**

*（部署后替换为你的 Vercel 链接）*

## ✨ 功能特性

- 🎯 **火炮计算器** - 精确计算迫击炮射程、高度和弹道
- 🗺️ **地图系统** - 基于 Leaflet 的战术地图
- 📊 **图层信息** - 显示阵地、载具生成点、攻击路线等
- 🔧 **工具集** - 方位角、距离、高度差等计算
- 🎮 **游戏同步** - 支持 Squad Mortar Overlay

## 🚀 部署说明

本项目已部署到 **Vercel** 平台，享受全球 CDN 加速和自动部署。

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start
```

### 构建生产版本

```bash
npm run build
```
